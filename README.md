# Учёт дневного стационара 1С (DayHospital-1C)

Конфигурация 1С для автоматизации дневного стационара медицинского учреждения: учёт пациентов, их пребывания в палатах, оказанных медицинских услуг, расхода медикаментов и питания.

## О проекте

Конфигурация охватывает полный цикл работы дневного стационара:

- **Приём и пребывание пациента** — оформление приёма в дневной стационар, размещение по палатам и отделениям, отслеживание сроков пребывания, выписка.
- **Медицинские услуги** — учёт оказанных пациентам медицинских услуг и расчёт стоимости пребывания.
- **Медикаменты и питание** — учёт лекарственных препаратов и питания, их расход и списание по пациентам.
- **Аналитика** — отчёт по загрузке отделений/палат и доходам учреждения.

Исходники выгружены штатным механизмом платформы (**Конфигурация → Выгрузить конфигурацию в файлы...**) в текстовом/XML-виде, что позволяет вести историю изменений в git без хранилища конфигурации (актуально для базовых версий 1С, где хранилище недоступно).

## Состав конфигурации

### Справочники

| Справочник | Назначение |
|---|---|
| **Пациенты** | Карточки пациентов |
| **Пол** | Справочник пола пациента |
| **Палаты** | Палаты стационара |
| **ОтделениеСтационара** | Отделения стационара |
| **МедицинскиеУслуги** | Перечень оказываемых медицинских услуг |
| **ЛекарственныеПрепараты** | Номенклатура лекарственных препаратов |
| **Питание** | Номенклатура рационов/позиций питания |
| **Сотрудники** | Карточки сотрудников учреждения |

### Документы

| Документ | Назначение |
|---|---|
| **ПриемВДневнойСтационар** | Оформление приёма пациента в дневной стационар |
| **ПребываниеПациента** | Учёт пребывания пациента (размещение, сроки) |
| **ВыпискаПациента** | Оформление выписки пациента |
| **РасходМедикаментов** | Списание/расход медикаментов на пациента |
| **РасходПитания** | Списание/расход питания на пациента |
| **СписаниеМедикаментовИПитания** | Общее списание медикаментов и питания со склада |

### Регистры и расчёты

| Объект | Назначение |
|---|---|
| **РасчетСтоимостиПребывания** | Расчёт стоимости пребывания пациента в стационаре |

### Отчёты

| Отчёт | Назначение |
|---|---|
| **ОтчетПоЗагрузкеИДоходам** | Отчёт по загрузке палат/отделений и доходам учреждения |

## Требования

- Платформа 1С:Предприятие 8.3 (для открытия и редактирования конфигурации в Конфигураторе)

## Как обновить репозиторий после изменений в конфигурации

1. Откройте базу в режиме **Конфигуратор**.
2. Выполните **Конфигурация → Выгрузить конфигурацию в файлы...**, указав эту же папку.
3. Зафиксируйте изменения:

```bash
git add .
git commit -m "Описание изменений"
git push
```

## Как развернуть конфигурацию из репозитория

1. Создайте новую информационную базу в 1С (или используйте существующую).
2. В Конфигураторе выполните **Конфигурация → Загрузить конфигурацию из файлов...**, указав путь к этой папке.
3. Обновите конфигурацию базы данных (**F7**).
